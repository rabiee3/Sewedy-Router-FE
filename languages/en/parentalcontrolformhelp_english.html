Parental Control

Enable

Enable or disables the Rule.

MAC Address

MAC Address of a LAN host device.

Start Time & End Time

Internet access to the LAN host device with MAC address specified will be controlled between "Start Time" and "End Time" based on the "Target" specified.

Days Of the Week

Internet access to the LAN host device with the MAC address specified will be controlled on the "Days Of The Week" based on the "Target" specified.

Target

Allow or Deny the traffic to the LAN host device.

E.g: With the following rule configuration, internet to the device with MAC "00:14:2a:3d:11:24" will be dropped between 03:00 AM to 03:00PM on Wednesday --
Enable - True
MAC Address - 00:14:2a:3d:11:24
Start Time - 03:00
End Time - 15:00
Days Of The Week - Wed
Target - Drop
