Device Management

Device Management allows to view and configure parameters relating to the CPE's association with an ACS

CWMP Enable Enables and disables the CPE's support for CWMP
ACS URL URL for the CPE to connect to the ACS
ACS Username Username used to authenticate the CPE when making a connection to the ACS
ACS Password Password used to authenticate the CPE when making a connection to the ACS
Periodic Inform Enable Whether or not the CPE MUST periodically send CPE the Inform method to the ACS
Periodic Inform Interval The interval(seconds) at which the CPE MUST attempt to connect with the ACS using Inform method if Periodic Inform Enable is true
Periodic Inform Time An absolute time reference in UTC to determine when the CPE will initiate the periodic Inform. Each Inform call MUST occur at this reference time plus or minus an integer multiple of the Periodic Inform Interval
Connection Request URL URL for an ACS to make a Connection Request to the CPE
Connection Request Username Username used to authenticate the ACS making a Connection Request to the CPE.
Connection Request Password Password used to authenticate the ACS making a Connection Request to the CPE.
Upgrades Managed Indicates whether or not the ACS will manage upgrades for the CPE.
CWMP Retry Minimum Wait Interval The lower limit for first session retry time range. The Inform is sent at a random time between the wait interval and the time calculated using the multiplier
CWMP Retry Interval Multiplier The multiplier for calculating upper time limit of first session retry
