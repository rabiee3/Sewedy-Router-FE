WiFi settings

Configure SSID or network name to operate in 5GHz range.It offers less interference and is ideal for media streaming and gaming

Activate band Enable or Disable WiFi 5GHz Radio
Country Select the Country (Regulatory domain).This will affect output power
Channel Radio channel for wireless connection operation
Operation mode ac-Only 802.11ac devices can connect,an-Only 802.11an devices can connect,a-Only 802.11a devices can connect
Channel Bandwidth Select the bandwidth of the channel. The higher the bandwidth, the higher the possible data rate.
Secondary Channel Select secondary extension channel position ( AboveControlChannel,BelowControlChannel,Auto) , applicable when operating in wide channel mode (i.e. when Channel Bandwidth is set to 40MHz,80MHz,120MHz or Auto).
AP Isolation Enabling this parameter will prevent clients to communicate with each other
SSID Unique name that identifies the wireless network. It is a string, and can can be up to 32 characters.
Advertise SSID Disabling this will hide SSID in the client's scan list. Client should have prior knowledge of the SSID name configured in order to connect
Security type Authentication method for wireless client to connect

Security type details

WPA-Personal,WPA2-Personal,WPA-WPA2-Personal,WPA2-Enterprise,WEP-64,WEP-128,None are the options under WiFi security type.The Security settings allows you to select the desired key type and allows setting the key value either in ASCII or HEX format. In WEP type, HEX format and in Passphrase, ASCII format is supported.
