ATM OAM F5 Loopback Diagnostic Test

This page allows you to do ATM OAM F5 Loopback Pings to check the VPI/VCI connections.

Click on:-

button under Actions and fill relevant fields to carry the test.

By default, the page shows previous test results. If the results shows blank or zeros, then no tests where conducted earlier.

Fields to fill in for the test

ATM PVCSelect ATM VPI/VCI from the drop down list. You can add new ATM PVC's from Internet Connections page.
Number of ATM OAM Cells to sendSpecify total number of ATM OAM Cells to send for test.
TimeoutTotal timeout (in ms) to complete the test. Optimal cases, use 300ms or more.

Results

Diagnostic StateShows "Complete" if the test completed successfully and "Error" if there is a failure.
Number of ATM OAM Cells ReceivedShows total number of ATM OAM Cells received.
Number of ATM OAM Cells FailedShows total number of ATM OAM Cells failed.
Average Response TimeAverage response time of the server sending back the OAM Cells (in ms).
Minimum Response TimeMinimum response time of the server sending back the OAM Cells (in ms).
Maximum Response TimeMaximum response time of the server sending back the OAM Cells (in ms).
