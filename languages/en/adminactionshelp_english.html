Auto Firmware Upgrade Settings

This page allows you to view and configure Software Upgrade parameters.

Auto-Upgrade
  • Disable: To disable automatic upgrade

  • Notify: Check for new firmware availability

  • Notify and upgrade: Check for new firmware and if available download and upgrade
    		•
    Server URL Provide server URL where the new firmware is hosted
    Start Time Provide the time to check new firmware
    Retry Count
  • If software upgrade fails due to server is unreachable, provide number of tries to reattempt software upgrade

  • Retry is applicable for Notify and Notify and upgrade both

  • Provide 0 if you wish to disable retry
    		•
    Retry Timer Time before the next retry is attempted.
