Classifier

You can configure and manage the Classifiers for IP-QoS, Where we can classify various kinds of traffics flows,based on Protocol headers/fileds

Classifier Name Give the name to Classifer
Order shows current order of classifier
Classifier Interface Select the name to Classifer interface (Incoming traffic)
Queueing Interface Select the name of the Queueing interface
Enable Shows current status of classifier, can be changed to Enable or disable
MPTCP Mark the traffic for mptcp processing
Queue Id Select the Queue for this classfied traffic
Layer - 2
Classification based on Layer - 2 protocol fields
Source MACClassifies traffic based on Source MAC Address
Source MAC MaskClassifies traffic based on Source MAC Mask Address
Source MAC ExcludeCurrent Inclusion/Exclusion status of Source MAC Address,can be changed to On or Off
Dest MACClassifies traffic based on Destination MAC Address
Dest MAC MaskClassifes traffic based on Destination MAC Mask Address
Dest MAC ExcludeCurrent Inclusion/Exclusion status of Dest MAC Address,can be changed to On or Off
VLAN Id RangeClassifes traffic based on given VLAN ID Range
VLAN Id ExcludeCurrent Inclusion/Exclusion status of VLAN ID,can be changed to On or Off
Incoming 802.1PClassifes traffic based on VLAN priority
ExcludeCurrent Inclusion/Exclusion status of VLAN priority,can be changed to On or Off
Layer - 3
Classification based on Layer - 3 protocol fields
Destintion IP v4/v6Classifies traffic based on Destination IP Address
Destination MaskClassifes traffic based on Destination IP Mask Address
Destination IP ExcludeCurrent Inclusion/Exclusion status of Dest IP Address,can be changed to On or Off
Source IP v4/v6Classifies traffic based on Source IP Address
Source MaskClassifes traffic based on Source IP Mask Address
Source IP ExcludeCurrent Inclusion/Exclusion status of Source IP Address,can be changed to On or Off
Layer - 4
Classification based on Layer - 4 protocol fields
Incoming DSCPClassifies traffic based on Incoming DSCP filed value
L4 ProtocolClassifies traffic based on Layer-4 protocol
ExcludeCurrent Inclusion/Exclusion status of L4 protocol, can be changed to On or Off
Dest Port Start Range Classifies traffic based on Layer-4 protocol's Destination Port Start address
Dest Port End Range Classifies traffic based on Layer-4 protocol's Destination Port End address
Dest Port ExcludeInclusion/Exclusion status of L4 protocol's Destination Port, can be changed to On or Off
Source Port Start Range Classifies traffic based on Layer-4 protocol's Source Port Start address
Source Port End Range Classifies traffic based on Layer-4 protocol's Source Port End address
Source Port ExcludeInclusion/Exclusion status of L4 protocol's Source Port, can be changed to On or Off
TCP AckClassifies TCP Ack's, Inclusion/Exclusion status, can be changed to On or Off
Layer - 7
Classification based on Layer - 7 protocols
Ex: FTP, SSH, Telnet, HTTP etc,.
Actions
Traffic Modification actions for classifier
Outgoing DSCPModifies the Classified traffic DSCP filed value
Outgoing 802.1PModifies the Classified traffic VLAN priority field value
Accelaration DisableCurrent PPA Accleration status , can be changed to Enable or Disable
Rate Limit (in kbps)Limits the Classified traffic rate
