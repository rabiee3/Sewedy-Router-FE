User Management Configuration

Enable

Enable/Disable the user

UserName

Username for the user. Username should be unique

Password

Password for the user

Role

This will specify the role of the user added. By default all the new user will be added as admin role. Role super_admin is root linux user and Role admin is Normal Linux User

Web Access

Enable/Disable user from accessing the CPE web

System, SSH & Telnet Access

Enable/Disable user from system, SSH and telnet login. SCP functionality will be enabled only on enabling system access.

FTP Access

Enable/Disable user from FTP login

Samba Access

Enable/Disable user from Samba login
