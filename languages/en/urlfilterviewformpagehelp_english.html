URL Filter Rule

Add or edit URL filter rules for blocking WEB access.

URL Configure URL to block WEB access

Example: Blocked domains - This could be "www.example.com". With this exmaple websites with exact match with domain will be blocked. where as www.example.net(if valid) will work.

Example: Wildcard - This could be "www.example*". With this exmaple websites matching example will be blocked, like www.example.com, www.example.net etc. wildcard entries in the middle of URLs will not be allowed, examples like goo*.com

Example: Substring match - This could be "exam". With this exmaple websites having this string in the domain name will be blocked.

Valid URL formats allowed

1. Actual domain name should start with atleast 2 alpha numeric characters. Example:"ex*", "ex...com", "www.ex*", "https://ex*", "https://www.ex*" and etc.

2. Actual domain name starting with special characters are not allowed. Example:"*example", "-example", ".example", "www.*example.com", "https://*example.com", "https://www.-example.com" and etc.

3. URLs ending with special characters(except '*') are not allowed. Example:"example.", "example-", "www.example.", "https://example-", "https://www.example-" and etc.

4. No Special characters allowed in URL, except '.', '-', '*'.

5. Example: "ex...com" will block "exam.com", "exin.com" and etc.
