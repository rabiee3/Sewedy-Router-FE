MODE Setting

DSL(Digital Subscriber Line) offers WAN DSL Connectivity on various DSL Modes. Provides configuration for xDSL modes, various annex modes and upstream and downstream attributes.

Default Values

DSL-Phy-TC Auto
UpStream Retransmission Enabled
DownStream Retransmission Enabled
Vectoring Enabled in Auto mode
Annex Modes G.dmt, ADSL, ADSL2, ADSL2 AnnexL, ADSL2+, VDSL

Note

On selecting DSL-Phy-TC as “Auto”, the CPE can auto negotiate either on VDSL PTM or ADSL ATM modes depending on the CO configuration

On selecting DSL-Phy-TC as ADSL AUTO or VDSL AUTO , CPE will negotiate in ATM/PTM mode of ADSL or VDSL modes respectively depending upon the CO Configuration

Selecting ADSL ATM/PTM/AUTO in PHY-TC configuration makes VDSL2 selection in Modes configuration obsolete

Selecting any ADSL Annexures in Modes configuration is obsoleted on configuring VDSL ATM/PTM/Auto in PHY-TC Configuration
