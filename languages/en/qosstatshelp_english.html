QosStats

Queue statistics table. This table create entries only for those {Queue, Interface} combinations for which statistics are to be collected

Name Shows the object name
Queue shows the path name of a row in the Queue table
Interface Shows the egress interface for which this Queue contains statistics
Output Packets Shows the number of packets output through the queue
Output Bytes Shows the number of bytes output through the queue
Dropped Packets Shows the number of packets dropped by the queue
Dropped Bytes Shows the number of bytes dropped by the queue
Queue Occupancy Packets Shows Queue occupancy in packets (gives a measure of queue latency)
Queue Occupancy Percentage Shows Queue occupancy measured as a percent, i.e. 100 * queue occupancy in bytes / queue size in bytes
