Multi-Brücke

Auf dieser Seite können Sie mehrere Bridges für Ihre Box konfigurieren.

Hinzufügen einer Brücke in Ihrem System

Um eine neue Brücke im System hinzuzufügen,

1. Klicken Sie auf die Schaltfläche Hinzufügen

2. Wählen Sie die Ports aus, die Sie der Bridge hinzufügen möchten

3. Klicken Sie auf Anwenden

Eine Brücke von Ihrem System ändern

Um eine Brücke im System zu ändern,

1. Klicken Sie auf die Schaltfläche Bearbeiten, die der Brücke entspricht

2. Deaktivieren Sie die Ports, die Sie von der Bridge entfernen möchten

3. Wählen Sie die Ports aus, die Sie der Bridge hinzufügen möchten

4. Klicken Sie auf Anwenden

Eine Brücke von Ihrem System entfernen

Um eine Brücke aus dem System zu entfernen, klicken Sie auf die Schaltfläche Löschen, die der Brücke entspricht.
