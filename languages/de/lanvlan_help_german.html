LAN-VLAN

Auf dieser Seite können Sie VLAN auf LAN-Schnittstellen in Ihrer Box hinzufügen / entfernen.

Hinzufügen eines VLAN auf der LAN-Schnittstelle in Ihrem System

Zum Hinzufügen eines VLANs im LAN, das über eine Schnittstelle im System vorhanden ist,

1. Klicken Sie auf Hinzufügen,

2. Wählen Sie die LAN-Schnittstelle aus dem Dropdown-Menü aus,

3. Stellen Sie die VLAN-ID bereit.

4. Klicken Sie auf Übernehmen, um die Änderungen auf das System anzuwenden.

Eine Brücke von Ihrem System entfernen

Um ein VLAN vom System zu entfernen, klicken Sie auf die Schaltfläche Löschen, die dem VLAN entspricht.
