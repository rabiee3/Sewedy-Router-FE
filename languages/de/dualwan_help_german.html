Multi-WAN

Auf dieser Seite können Sie Multi-WAN für Ihre Box konfigurieren.

Erstellen einer Multi-WAN-Gruppe

Folgen Sie den unten aufgeführten Schritten, um eine Multi-WAN-Gruppe im System zu erstellen,

1. Klicken Sie auf Hinzufügen.

2. Wählen Sie die WANs aus, die Sie unter Multi-WAN-Gruppe hinzufügen möchten. Im Fall des FailOver-Modus werden WANs entsprechend der Reihenfolge, in der WANs unter Multi-WAN-Gruppe hinzugefügt werden, mit Priorität zugewiesen. Beispiel: Das zuerst hinzugefügte WAN hat die höchste Priorität, das zuletzt hinzugefügte WAN hat die niedrigste Priorität.

3. Füllen Sie Multi-WAN-spezifische Konfigurationsparameter auf,

Multi-WAN-spezifische Konfigurationen
ParameterStandardwertBeschreibung
Ziel IP KeinerPassen Sie den Verkehr an die angegebene Zieladresse an, für die Multi-WAN-Funktionalität benötigt wird.
Politik KeinerMulti-WAN-Richtlinie, Loadbalanced / FailOver

4. Füllen Sie WAN-spezifische Konfigurationsparameter auf

Entfernen einer Multi-WAN-Gruppe

Um eine Multi-WAN-Gruppe aus dem System zu entfernen, klicken Sie auf die Schaltfläche Löschen.

Für den "Failover" -Modus hängt die Priorität von der Reihenfolge ab, in der WANs der Multiwan-Gruppe hinzugefügt werden.
