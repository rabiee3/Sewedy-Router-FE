LAN

Local Area Network ( LAN) bereitgestellt Vernetzbarkeit Clients mit dem LAN verbunden . LAN ermöglicht gemeinsame Nutzung von Ressourcen , Datenaustausch, Informationssicherung und die Arbeitsverteilung zwischen den Geräten . Muss jedes LAN eine eindeutige Internet -Protokoll-Adresse (IP) eindeutig identifiziert und mit beliebigen Vorrichtung Standardwerte für die einzelnen Felder ist zu kommunizieren , wie folgt:

IP Adresse 192.168.1.1
Subnet Mask 255.255.255.0
beginnen Adresse 192.168.1.2
End Anschrift 192.168.1.254
Lease Time 1 Day

Die Start -IP und End IP sollte im Subnetz Bereich von IP- Adresse und Subnet- Kombination sein,

Example

Wenn die IP-Adresse -192.168.2.1 wird die IP beginnen dann End IP sollte in 192.168.2.x subnet.If Subnet Mask sein ist 255.255.255.128 sollte End IP unter 192.168.2.126 sein
