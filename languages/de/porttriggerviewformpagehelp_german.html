Port Auslöser-Regel

Hinzufügen oder Bearbeiten von Regeln zum dynamischen Weiterleiten eines bestimmten Anschlusses an sich selbst (DUT).

Aktivieren Aktiviert den Eintrag Regel
Auslöser Port Port für ausgehenden Verkehr
Auslöser Portbereich Portbereich für ausgehenden Datenverkehr. Wenn er '0' ist, wird dieser Parameter ignoriert.
Auslöser Protokoll Protokoll für ausgehenden Verkehr
Eingehend Port Port für eingehenden Datenverkehr
Eingehend Portbereich Portbereich für eingehenden Datenverkehr. Wenn er '0' ist, wird dieser Parameter ignoriert.
Eingehend Protokoll Protokoll für eingehenden Datenverkehr

Beispiel: Mit der folgenden Konfiguration wird ftp-Verkehr nur dann an den Host weitergeleitet, wenn ssh-Verkehr von einem Host initiiert wird. Wenn kein ssh-Verkehr eingeleitet wird, wird kein FTP-Verkehr an den Host weitergeleitet.

Aktivieren TRUE
Auslöser Port 22
Auslöser Portbereich 0
Auslöser Protokoll TCP
Eingehend Port 21
Eingehend Portbereich 0
Eingehend Protokoll TCP
