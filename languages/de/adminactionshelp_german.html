Auto Firmware Upgrade-Einstellungen

Auf dieser Seite können Sie Software Upgrade-Parameter anzeigen und konfigurieren.

zu überprüfen
Auto-Upgrade
  • Deaktivieren: Automatisches Upgrade deaktivieren

  • Benachrichtigung: Auf neue Firmware-Verfügbarkeit prüf

  • Benachrichtigen und aktualisieren: Auf neue Firmware prüfen und falls vorhanden Download und Upgrade
    		•
    Server URL Server-URL angeben, auf der die neue Firmware gehostet wird
    Startzeit Geben Sie die Zeit, um neue Firmware
    Wiederholen Sie Count
  • Wenn Software-Upgrade fehlschlägt, da Server nicht erreichbar ist, geben Sie die Anzahl der Versuche an, Software-Upgrade erneut zu versuchen

  • Wiederholen gilt für Benachrichtigung und Benachrichtigung und aktualisieren beide.

  • Geben Sie 0 an, wenn Sie die Wiederholung deaktivieren möchten
    • Wiederholungs-Timer Zeit, bevor der nächste Wiederholversuch versucht wird.
