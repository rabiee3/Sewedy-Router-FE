VoIP

Auf dieser Seite können Sie VoIP konfigurieren.

Proxy Adresse Proxy-Server-Adresse. Dies kann entweder eine IP-Adresse oder ein Domänenname sein.
Registrar Adresse Adresse des Registrarservers. Dies kann entweder eine IP-Adresse oder ein Domänenname sein.
Domänenname Domain-Adresse. Dies kann entweder eine IP-Adresse oder ein Domain-Name sein.
Status Status der Standardzeile.
Status der Leitungsregistrierung Registrierungsstatus der Standardzeile.
Anzeigename SIP Anzeigename zum Konfigurieren der Standardleitung.
Nutzername SIP-Benutzername zum Konfigurieren der Standardleitung.
Passwort SIP Passwort zum Konfigurieren der Standardleitung.
